## 2. 架构

### 2.1 文件结构

```
webapp/src/
├── i18n/
```

### 2.2 技术栈

| 层 | 技术 | 说明 |
|----|------|------|
| 状态管理 | Pinia（`themeStore`） | 语言状态与主题状态合并在同一 Store |
| 翻译函数 | `createTranslator()` | 支持点号路径 + 参数替换 |
| 响应式 | Vue `computed()` | 语言切换后自动更新所有翻译文本 |
| 持久化 | `localStorage` | key: `allfi-language` |

## 3. 核心实现

### 3.1 语言配置（`themeStore.js`）

```javascript
export const languages = [
  { code: 'zh-CN', name: '简体中文', flag: '🇨🇳', short: '中' },
  { code: 'zh-TW', name: '繁體中文', flag: '🇹🇼', short: '繁' },
  { code: 'en-US', name: 'English', flag: '🇺🇸', short: 'EN' }
]
```

Store 提供的语言相关状态和方法：

| 字段/方法 | 类型 | 说明 |
|-----------|------|------|
| `currentLanguageCode` | `ref` | 当前语言代码 |
| `currentLanguage` | `computed` | 当前语言配置对象 |
| `availableLanguages` | `computed` | 所有可用语言列表 |
| `setLanguage(code)` | 方法 | 切换语言并持久化 |
| `initLanguage()` | 方法 | 从 localStorage 恢复语言设置 |

### 3.2 翻译函数（`i18n/index.js`）

```javascript
// 创建翻译器
export function createTranslator(langCode) {
  const dict = translations[langCode] || translations['zh-CN']

  return function t(key, params = {}) {
    // 支持点号路径：t('settings.language')
    const keys = key.split('.')
    let value = dict
    for (const k of keys) {
      if (value && typeof value === 'object' && k in value) {
        value = value[k]
      } else {
        console.warn(`Translation missing: ${key} for ${langCode}`)
        return key  // 缺失翻译返回 key 本身
      }
    }
    // 支持参数替换：t('common.pageInfo', { current: 2, total: 10 })
    let translated = value
    for (const paramKey in params) {
      translated = translated.replace(`{${paramKey}}`, params[paramKey])
    }
    return translated
  }
}
```

### 3.3 Vue Composable（`composables/useI18n.js`）

```javascript
import { computed } from 'vue'
import { useThemeStore } from '../stores/themeStore'
import { createTranslator, translations } from '../i18n/index'

export function useI18n() {
  const themeStore = useThemeStore()
  const locale = computed(() => themeStore.currentLanguageCode)
  const translator = computed(() => createTranslator(locale.value))

  const translate = (key, params) => translator.value(key, params)

  const getTranslations = (section) => {
    const dict = translations[locale.value] || translations['zh-CN']
    return dict[section] || {}
  }

  return {
    t: translate,         // 翻译函数
    locale,               // 当前语言代码（响应式）
    getTranslations       // 获取整个分区的翻译对象
  }
}
```

## 4. 使用方法

### 4.1 在组件中使用翻译

```vue
<script setup>
import { useI18n } from '../composables/useI18n'

const { t } = useI18n()
</script>

<template>
  <div>
    <h1>{{ t('dashboard.title') }}</h1>
    <p>{{ t('dashboard.totalAssets') }}</p>
    <button class="btn btn-primary">{{ t('common.save') }}</button>
  </div>
</template>
```

### 4.2 参数化翻译

翻译定义：
```javascript
pageInfo: '第 {current} 页，共 {total} 页'
```

使用：
```javascript
t('common.pageInfo', { current: 2, total: 10 })
// → "第 2 页，共 10 页"
```

### 4.3 语言切换（Settings.vue）

```vue
<select
  :value="themeStore.currentLanguageCode"
  @change="themeStore.setLanguage($event.target.value)"
  class="input select setting-select"
>
  <option
    v-for="lang in languageOptions"
    :key="lang.value"
    :value="lang.value"
  >
    {{ lang.label }}
  </option>
</select>
```

### 4.4 获取整个分区翻译

```javascript
const { getTranslations } = useI18n()
const themeNames = getTranslations('themes')
// → { 'nexus-pro': { name: 'Nexus Pro', ... }, ... }
```

## 6. 持久化机制

```javascript
// themeStore.js
const LANGUAGE_STORAGE_KEY = 'allfi-language'

function setLanguage(langCode) {
  const lang = languages.find(l => l.code === langCode)
  if (!lang) return false

  currentLanguageCode.value = langCode
  localStorage.setItem(LANGUAGE_STORAGE_KEY, langCode)  // 持久化
  document.documentElement.lang = langCode               // 设置 HTML lang 属性

  return true
}

function initLanguage() {
  const saved = localStorage.getItem(LANGUAGE_STORAGE_KEY)
  if (saved && languages.find(l => l.code === saved)) {
    currentLanguageCode.value = saved
  }
  document.documentElement.lang = currentLanguageCode.value
}
```

**特性**：
- 切换立即生效，无需刷新页面
- `document.documentElement.lang` 同步更新（利于 SEO 和辅助功能）
- 应用初始化时自动恢复用户选择

## 8. 翻译规范

### 8.1 Key 命名

- 使用语义化名称：`dashboard.totalAssets`（推荐） vs `text1`（禁止）
- 按功能模块分组，与页面/组件对应
- 嵌套不超过 3 层

### 8.2 翻译一致性

```javascript
// 同一概念保持统一用词
'zh-CN': { save: '保存' }     // 全局统一
'zh-TW': { save: '儲存' }     // 繁体对应
'en-US': { save: 'Save' }     // 英文对应
```

### 8.3 参数化翻译

动态内容使用 `{param}` 占位符：
```javascript
txSyncSuccess: '成功同步 {count} 条交易记录'
// 使用: t('settings.txSyncSuccess', { count: 42 })
```
